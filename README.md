# DAIBot
:: 디스코드 봇 커스텀

키워드를 한개 또는 여러개(콤마로 구분) 입력하여 검색하면
상위 3개 결과물의 제목, 작성자, 링크를 답변함
(쇼핑은 5개)

discord slash command 추가
    1. "/유튜브 "
        유튜브 동영상 검색
    2. "/뉴스 "
        네이버 뉴스 검섹
    3. "/블로그 "
        네이버 블로그 검색
    4. "/카페 "
        네이버 카페 검색
    5. "/쇼핑 "
        네이버 쇼핑 검색
    6. "/로또 "
        키워드에 '당첨번호' 또는 '추천번호'를 입력
        최근 당첨번호 확인 및 추천번호 생성
